## 원래 프로젝트에서 개선된 사항
- 난이도 선택 추가
- 일시정지 기능 추가
- 게임시작 버튼 추가
- 다시하기 기능 추가
- 소리 추가, 음소거 기능 추가
- 효과음 기능 여러개 재생하는 알고리즘 
- 시간 측정 기능 추가
- 동적 카드 생성 => 숫자 만 입력하면 카드가 자동으로 생성
- grid repeat 속성을 이용해서 각 열과 행 만큼 repeat 해줌으로 행과 열을 생성
- 행과 열의 곱을 총 카드개수로 구함
- 총 카드 개수로 카드 생성
- 동적 요소 생성
- 중복 코드 제거 => 함수로 만들기, 랜덤 함수, 사운드 함수, gamestart
- 코드 주석 추가

### 🎮 게임구현 알고리즘
- classList로 카드 앞 뒷면 구분 => flipped가 있다면 뒷면( rotateY(180deg) )
- classList에 flipped가 있다면 카드가 뒷면으로 뒤집어 진다.
- classList에 flipped가 없다면 카드가 앞면으로 뒤집어 진다. 
- 카드를 클릭하면 해당 카드를 카드배열에 넣는다. 
- 클릭된 카드는 다시 클릭되지 않도록 한다.
- 배열에 담은 카드가 2개 일 때, 각각의 카드의 name이 같은지 비교해서 같다면 완료카드 배열에 넣고, 
- 해당 카드는 더 이상 클릭이 되지 않도록 유지한다.
- 카드가 서로 일치하지 않으면 카드를 다시 classList filpped 추가하여 뒤집는다. 카드가 다시 클릭 되도록 한다.
- 위 알고리즘을 반복하면서 최종 완료카드의 배열의 길이가 총 카드개수와 일치할때 게임을 종료시킨다.
